AI Generative Website

Overview

AI Generative website with Text, Code, Image, Video, and Music Generation features. Using OpenAI API for the Text, Code and Image generations and Replicate API for Video and Music generations. Payment handled by Stripe services and Prisma that stores the database of the user in order to count the free generations of the users and checks if user is using premium subscription.

See Demo Source Code

Main Tech Stack Used: Next.JS, React, Tailwind, Stripe, Prisma
